# Smart-Expense-Tracker-Java

## Project Overview
Smart Expense Tracker is a console-based Java application designed to help users manage and track their daily expenses efficiently.  
The project is built using **Object-Oriented Programming (OOP)** principles and **file handling** to ensure data persistence and structured code design.

This application helps users understand their spending habits and avoid overspending by providing budget alerts and category-wise expense reports.

---

## Features
- Add daily expenses with date, category, amount, and notes
- View all recorded expenses
- Generate category-wise expense reports
- Set a monthly budget
- Get alerts when the budget limit is exceeded
- Persistent storage using file handling (data is saved and loaded automatically)

---

## Technologies Used
- Java  
- Object-Oriented Programming (OOP)  
- File Handling  
- ArrayList  
- Console-based input/output
